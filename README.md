# 经典小游戏合集

这是一个包含多个经典小游戏的网页游戏合集，所有游戏都可以直接在浏览器中运行。

## 🎮 游戏列表

1. **超级玛丽** (`mario.html`) - 平台跳跃游戏
2. **俄罗斯方块** (`tetris.html`) - 经典益智消除游戏
3. **贪吃蛇** (`snake.html`) - 经典贪吃蛇游戏
4. **打砖块** (`breakout.html`) - 打砖块游戏
5. **飞机大战** (`invaders.html`) - 射击游戏
6. **推箱子** (`sokoban.html`) - 益智推箱子游戏
7. **2048** (`game2048.html`) - 数字合并游戏

## 🚀 如何访问

### 方法1：通过 GitHub Pages（推荐）

1. 进入 GitHub 仓库：https://github.com/frontzhm/min-games
2. 点击仓库右上角的 **Settings（设置）**
3. 在左侧菜单中找到 **Pages**
4. 在 **Source** 部分选择 **Deploy from a branch**
5. 选择分支：**main**
6. 选择文件夹：**/ (root)**
7. 点击 **Save** 保存
8. 等待几分钟后，访问：`https://frontzhm.github.io/min-games/`

### 方法2：本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 📝 使用说明

- 所有游戏都支持键盘操作
- 每个游戏都有详细的操作说明
- 游戏数据会自动保存到本地存储

- ✅ 纯 HTML/CSS/JavaScript 实现，无需服务器
